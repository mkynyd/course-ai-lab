## 账号与注册

**问:注册需要哪些信息?**

答:邮箱、密码、注册码。当前版本没有"开发模式绕过",本地开发也需要有效的注册码。

**问:注册码忘记填了怎么办?**

答:重新发起注册请求即可,旧的注册失败不消耗注册码。

**问:如何重置密码?**

答:当前版本暂未提供自助重置流程,需联系管理员重置。

**问:Provider API Key 在哪里配置?**

答:不配置在本地 `.env` 中,而是由管理端通过注册码/Credential Profile 集中下发。

## 聊天与模型

**问:为什么我的提问总是落到"通用问答"?**

答:当前生产路径使用项目类型决定提示词模板,不会按关键词路由到具体任务类型。可在项目设置切换类型,或点击快捷任务按钮。

**问:深度推理模式有什么副作用?**

答:响应延迟上升、token 用量增加。适合复杂推理与多步问题。

**问:对话上下文最长支持多少?**

答:由模型窗口决定,应用层目前不会主动截断历史消息。

## 项目与资料

**问:项目类型可以中途切换吗?**

答:API 支持 `PATCH /api/projects/:id` 修改 `type`,但前端编辑 UI 尚未接入。

**问:上传文件最大多少?**

答:当前客户端和 API 均硬编码为单文件 20MB。

**问:扫描型 PDF 怎么识别?**

答:所有 PDF 均由 MiniMax M3 原生文档解析处理,不区分文本型/扫描型,也不使用 PDF.js。

**问:为什么我勾选的文件没有进入上下文?**

答:可能文件状态不是 `parsed`,或上下文长度超出限制。可在资料列表查看状态。

## 成果与导出

**问:Artifact 支持哪些类型?**

答:API 可接受约 13 种类型,如 `experiment_report`、`calculation`、`review_outline`、`code_explanation`、`mermaid`、`general` 等;UI 展示约 12 种标签。

**问:导出格式之间可以互转吗?**

答:可以,以 Markdown 作为唯一正文源导出。

**问:Artifact 导出的 ZIP 包含什么?**

答:Artifact 导出只有单文件(markdown/docx/pdf),没有 ZIP。带 `pics/` 的完整 ZIP 包属于 `/tools` 文档转换流程。

**问:导出失败如何排查?**

答:查看服务端日志,确认 PDFKit/Chromium/DOCX 依赖、`CHROMIUM_EXECUTABLE_PATH` 和对象存储配置。

## Agent 模式

**问:为什么我看不到 Agent 入口?**

答:当前没有独立 Agent 入口;Agent 事件会内联显示在对话中,MiniMax 路径当前不携带客户端工具。

**问:审批卡片一直等待,如何取消?**

答:点击 **拒绝**,或刷新页面。刷新会清空未完成审批,需重新发起对话。

**问:为什么 Tool 调用被 blocked?**

答:可能原因:Tool 未注册、权限 scope 缺失、Workspace 策略 block、Skill 白名单不匹配、风险等级超限、参数校验失败。详见 [Policy Engine](./architecture/policy-engine.md)。

**问:参数替换攻击是怎么被阻止的?**

答:审批令牌只存 `sha256(raw)`,兑换时重新校验 `argumentsHash`。模型在 `proposed` 与 `approved` 之间替换参数会被拒绝。

## 缓存与性能

**问:Redis 不可用怎么办?**

答:限流会降级为内存实现;导出缓存失效时直接重新生成,不会报错。

**问:为什么外部 API 缓存默认关闭?**

答:MiniMax active cache 与 prompt reorder 实验默认关闭,需在收集真实基线后再决定是否启用。

**问:长对话会卡顿吗?**

答:长对话已切换到 TanStack Virtual 虚拟化,仅渲染可视区域。流式中的最后一条消息保持直接渲染。

## 部署与运维

**问:生产环境必须用对象存储吗?**

答:是的。本地卷在多实例部署下会出现文件不一致,生产环境推荐七牛云 Kodo 私有 Bucket。

**问:Alpha 模式如何撤销用户?**

答:管理端显式撤销后,该用户的 `accessStatus` 会设为 `revoked`,新的 Provider 调用会失败,但当前登录会话不会自动登出。

**问:如何回滚数据库迁移?**

答:不推荐 `prisma migrate resolve --rolled-back`,应新建修复迁移,在新版本中修正。
